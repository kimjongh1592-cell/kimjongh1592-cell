# 소개

데이터 가공 및 파이프라인 구축, 그리고 생성형 AI 기반의 서비스 구현에 관심이 많은 학부생입니다. 복잡한 정형/비정형 데이터를 분석 가능한 형태로 정제하고, 이를 시각화하여 인사이트를 도출하는 과정에 흥미를 가지고 있습니다.

## 기술 스택 (Tech Stacks)
- Languages: Python, JavaScript, SQL
- Libraries & Tools: OpenCV, Pandas, Matplotlib, Streamlit, KNIME
- DevOps & Tools: GitHub Desktop, Git

## 주요 관심 연구 분야
- 컴퓨터 비전 및 가상 데이터(Synthetic Data) 품질 평가
- 멀티모달 학습 분석 파이프라인 및 연구 자동화 도구 개발
